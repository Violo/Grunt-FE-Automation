Boot up enviroment

Tasks & commands

  • compass watch (polling scss changes)
  • grunt compass (compress css)
  • grunt minifyHtml (compress html)
  • grunt validation (validate html)
  • grunt csslint (test css)
  • grunt svgmin (clean svg files)
  • grunt grunticon (create image assets and fallbacks)
  • grunt styleguide (create styleguide scaffolding)
  • grunt uncss (remove unused style)
  • grunt palettable (create used colors list)
  • pleeease compile (css postprocessor)

Registered Tasks

  • grunt build (all tasks: uglify, minifyHtml, validation, csslint:strict, svgmin, grunticon, gruntstyleguide, palettable, compass:dist)
  • grunt start (start tasks: uglify, minifyHtml, grunticon, copy, compass:dist)
  • grunt dev (development tasks: svgmin, grunticon, palettable, csslint:strict, compass:dev)
  • grunt serve (serving page in localhost:8080)

Scss Files (Custom SMACSS Categorization)

  • Define Rules: variables, mixin, operations: app.scss
  • Base Rules: (Reset from Framework) + base.scss
  • Layout Rules: layout.scss
  • State Rules: state.scss
  • Theme Rules: theme.scss
  • Modules Rules:
    • Completion Bar: module completionbar.scss
    • Thermomethers: module thermometers.scss
    • Popover: module popover.scss
    • Thouchspinner: module thouchspinner.scss

Styleguide

  • Images Assets (data-svg, data-png, fallback png): Images preview
  • CSS Assets: Scaffolding
  • Color Assets: Used colors
