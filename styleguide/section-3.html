3.0 - State Visibility

List of available visibility class

.hidden (Hide from both screenreaders and browsers: h5bp.com/u)
.visuallyhidden (Hide only visually, but have it available for screenreaders: h5bp.com/v)
.visuallyhidden.focusable:active (Allow the element to be focusable from keyboard: h5bp.com/p)
.visuallyhidden.focusable:focus (Allow the element to be focusable from keyboard: h5bp.com/p)
.invisible (Hide visually and from screenreaders, but maintain layout)
.unselectable (not selectable)
